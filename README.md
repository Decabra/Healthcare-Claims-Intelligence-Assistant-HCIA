# Healthcare-Claims-Intelligence-Assistant-HCIA-
Healthcare Claims Intelligence Assistant integrating Airflow ETL, dbt modeling, Postgres/pgvector, and a LangGraph RAG workflow served via FastAPI, combining automated pipelines, semantic retrieval, ICD-10/CPT ontologies, denial logic, and multi-step LLM reasoning to support natural-language Q&amp;A over 50k+ synthetic claims and clinical notes.
